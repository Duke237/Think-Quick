# Sound Files Needed

Place these sound files in this directory:

1. `timer-start.mp3` - Sound when timer starts
2. `timer-end.mp3` - Sound when timer completes
3. `correct.mp3` - Sound for correct answer
4. `wrong.mp3` - Sound for wrong answer (buzzer)
5. `strike.mp3` - Sound when strike is added

You can download free sounds from:
- https://freesound.org
- https://mixkit.co/free-sound-effects/
- https://soundbible.com
